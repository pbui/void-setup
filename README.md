Void Setup
==========

This is a collection of scripts for configuring [Void Linux]:

- `setup-profile.d.sh`:

    Installs `/etc/profile.d` scripts.

- `setup-syslinux.sh`:

    Installs and configures the [syslinux] bootloader.

[Void Linux]:   https://www.voidlinux.eu/
[syslinux]:     http://www.syslinux.org/
